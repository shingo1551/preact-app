# Preact-App

## vite

```bash
npm create vite@latest preact-app -- --template preact-ts
```

## preact-router

```bash
npm i preact-router
```

## tailwindcss

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```
